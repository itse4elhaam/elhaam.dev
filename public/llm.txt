## Technical Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **UI Components:** shadcn/ui
- **Language:** TypeScript 5.6
- **Content:** Markdown with frontmatter
- **Markdown Processing:** remark + rehype ecosystem
- **Hosting:** Vercel (Edge Network)
- **Theme:** Dark/Light mode with system preference

## Design Principles

1. **Simplicity over cleverness** — Clean, distraction-free reading experience
2. **Performance first** — Optimized Core Web Vitals
3. **Accessibility** — WCAG 2.1 compliant
4. **Responsive** — Mobile-first design approach

## Key Features

- **Theme Toggle:** Light/Dark/System preference detection
- **Text Selection Share:** Share quotes directly to Twitter
- **Reading Time:** Automatic calculation per article
- **SEO Optimized:** Full metadata, OpenGraph, Twitter Cards
- **AI-Ready:** llm.txt for LLM content parsing

## Content Metadata Structure

Each post includes:
- title: String
- description: String (meta description)
- date: ISO 8601 format
- category: Category slug
- tags: Array of strings
- readingTime: Calculated string

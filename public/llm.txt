## Content Categories

### 1. Engineering
Technical deep dives, architecture patterns, code craftsmanship, and system design.
- Topics: architecture, systems, design, programming, software engineering
- Depth: Comprehensive, 2000+ words per article
- Examples: "Building Thoughtful Architecture"

### 2. Personal
Thoughts on engineering life, career growth, team culture, and personal development.
- Topics: culture, teams, leadership, career, remote work
- Tone: Reflective, experience-based

- Topics: quick tips, observations, bookmarks
- Length: 500-1000 words
- Format: Short-form, actionable

## Technical Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **UI Components:** shadcn/ui
- **Language:** TypeScript 5.6
- **Content:** Markdown with frontmatter
- **Markdown Processing:** remark + rehype ecosystem
- **Hosting:** Vercel (Edge Network)
- **Analytics:** Vercel Analytics (free, privacy-focused)
- **Theme:** Dark/Light mode with system preference (next-themes)
- **Theme Toggle:** Light/Dark/System preference detection
